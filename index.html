Why ModBot?

📋
Persistent Mod Logs
Every action is stored in a local JSON file. The last 500 moderation events are always available, no database required.
⚡
Slash Commands
Fully integrated with Discord's slash command system — with autocomplete, type-safe parameters, and ephemeral responses.
🔒
Permission Checks
Every command verifies both the user's and bot's permissions before executing. No accidental escalation.
📬
DM Notifications
Users are notified by DM when they receive a warning, with full embed formatting and server name included.
⚠️
Warning System
Track, display, and clear warnings per-guild and per-user with full timestamps and reasons preserved.
🧩
Zero Config
Drop in your token and Guild ID and you're live. No external databases, no dashboard, no monthly fees.
